📌 Project Overview

The Unicorn Denmart Business Performance Dashboard is an end-to-end data analytics project that provides insights into orders, inventory, and delivery performance.

- The goal of this project is to help business stakeholders monitor operations, identify inefficiencies, and make data-driven decisions to improve overall performance.

🎯 Objectives
- Analyze sales and order trends
- Track inventory levels and stock issues
- Evaluate delivery performance and delays
- Identify top-performing products and locations
- Provide actionable insights for business optimization

🛠️ Tools & Technologies
- Power BI – Dashboard creation & visualization
- SQL – Data extraction and querying
- Python (Pandas, NumPy) – Data cleaning and preprocessing

📊 Key Features of Dashboard
🔹 Overview Dashboard
- Order counts trend over time
- Distribution of top 10 categories ordered
- Order Distribution through out India

🔹 Clinic Insights
- Revenue trends over time
- Clinis who order most frequently and quantity
- Order distribution by location
- Clinic vs Product category demand distribution

🔹 Inventory Analysis
- Current stock levels
- Stockout identification
- Product-wise inventory tracking

🔹 Delivery Performance
- Delivery time distribution
- Average delivery time by location
- Delivery trends over time

🔹 Product Performance
- Top products by quantity ordered
- Top Revenue generating products
- Top 4 revenue genrating products - Demand trend analysis

📈 Key Insights
- Identified high-demand products contributing most to revenue
- Detected stockout issues impacting sales
- Found delivery delays in specific locations
- Highlighted opportunities to optimize inventory and logistics
